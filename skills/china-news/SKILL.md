# 综合资讯技能（国内 + 国际）

同时抓取**政务与权威央媒**、**国内门户**和**国际中文媒体**的信息，覆盖：

- 💹 **经济政策** - 财政、货币、贸易、产业政策等
- 🌍 **地缘政治** - 国际关系、地区冲突、外交动态
- ⚔️ **战争事件** - 军事冲突、战争进展、武器装备
- 🏛️ **政治决策** - 立法、外交政策、重要会议与政务动态
- 📱 **科技数码** - AI、互联网、创业投资等

## 使用方法

```bash
python3 ~/.openclaw/skills/china-news/scripts/fetch_china_news.py [天数] [分类]
```

**参数说明：**
- `天数`：回溯天数（默认 `0` = **仅保留发布时间落在「今天 0 点起」且可解析日期的条目**；无发布时间的条目会丢弃。需要「滚动近 N 天」时用 `N`）
- `分类`：all | domestic | world | economy | geopolitics | war | politics | tech | security（默认 all）
- **抓取源数量上限**（可选其一）：
  - 环境变量：`export OPENCLAW_NEWS_MAX_SOURCES=12`（按脚本内置优先级只请求前 N 个源）
  - 命令行：`python3 fetch_china_news.py 3 all --max-sources 12`

**示例：**
```bash
# 今日全部资讯
python3 fetch_china_news.py

# 近3天地缘政治新闻
python3 fetch_china_news.py 3 geopolitics

# 近7天科技数码
python3 fetch_china_news.py 7 tech

# 国内动态
python3 fetch_china_news.py 0 domestic
```
